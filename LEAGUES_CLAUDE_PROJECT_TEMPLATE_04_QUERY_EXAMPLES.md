# Query Examples

Use these prompts directly in the Claude Project.

## Fact Queries

- `What magic tasks give the most points in Desert? Cite which uploaded doc you used.`
- `Which regions have the highest point totals and what are they best for? Cite the source doc.`
- `Summarize the most important relics by tier for a mage plan.`

## Build Queries

- `I want a lazy magic build. Recommend 3 regions, key relics, and the main tradeoffs.`
- `Compare Kandarin + Desert + Zeah against Kandarin + Desert + Asgarnia for magic.`
- `What are the strongest community-backed ranged archetypes right now?`

## Route Queries

- `I care about early points and low effort. Which regions and tasks should I prioritize first?`
- `Build me a dad-style route using the uploaded docs only.`
- `What is the safest early-to-mid game plan if I want strong magic later?`

## Validation Queries

- `Separate verified wiki/reference facts from community opinions in your answer.`
- `Tell me where the community disagrees instead of pretending there is one consensus.`
- `If the uploaded docs do not support a claim, say that clearly.`

## Planner Support Queries

- `Given this region trio, what relic direction best fits it and why?`
- `What are the main opportunity costs of locking this region combination?`
- `Summarize the current strongest high-confidence consensus in 8 bullets max.`

# Query Examples — Copy-Paste Prompts for Mobile / iOS

Use these prompts directly in this Claude Project on claude.ai or the iOS app.
Always add: "Cite which uploaded doc you used."

---

## Fact Queries

```
What magic tasks give the most points in Kandarin? Cite which uploaded doc you used.
```

```
What is the Thermonuclear Smoke Devil echo and why is it important for magic? Cite the source doc.
```

```
What does Grimoire do and why do magic builds take it instead of Desert? Cite the source doc.
```

```
Summarize all 8 relic tiers and the best pick at each tier for a magic build. Cite the source doc.
```

```
What is the Grotesque Guardians echo and why does Morytania matter for magic? Cite the source doc.
```

```
What are the key differences between Flask of Fervour and Minion at Tier 8? Cite the source doc.
```

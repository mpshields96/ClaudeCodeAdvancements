## Recommended Route Notes

### Phase 0 — Tutorial + First Hour (Varlamore)

- Complete tutorial, pick Magic pact at character creation
- Select Endless Harvest T1 — PERMANENT, no going back
- Immediately: start Gem Crab AFK for passive combat XP
- Early tasks in Varlamore: anything in task list that doesn't require high stats
- Unlock Perilous Moons as soon as 48 Slayer (38 with Abundance — available via Reloaded T7 later)
- Fish and gather passively — Endless Harvest doubles all resources immediately

### Phase 1 — Varlamore + Karamja early tasks (~0–600 LP)

**Goal:** Hit T2 relic threshold (600 LP)

- Stack easy/medium tasks in Varlamore while AFK
- Karamja: TzHaar for early combat XP, Dragon imps for points
- Fish at Barbarian Fishing in Kandarin as soon as you unlock it (AFK score 8; can do at work)
- Do NOT rush Perilous Moons before Slayer level is ready
- At 600 LP: pick Friendly Forager — herbs auto-bank from all skilling immediately

### Phase 2 — Unlock Kandarin (~600–1500 LP)

**Goal:** Unlock Kandarin, start building toward Thermy echo; hit T3 (1500 LP)

- Kandarin unlock priority: early tasks, Barbarian Fishing XP chain, Ranging Guild for points
- Thermonuclear Smoke Devil echo: kill it for BiS magic weapons — should be accessible early-mid game
- Barbarian Fishing: run this AFK at work every session
- At 1500 LP: pick Evil Eye — now you can teleport to any boss. Huge quality of life upgrade.

### Phase 3 — Unlock Kourend (~1500–2500 LP)

**Goal:** Unlock Kourend & Kebos; hit T4 (2500 LP)

- Kourend tasks: massive task density — work through easy/medium/hard systematically
- Catacombs of Kourend: start Slayer tasks here; Soul Harvest later converts these bones to Prayer XP
- Alchemical Hydra available when Slayer level ready (95 Slayer standard; check with Abundance bonus)
- CoX: unlock when comfortable; don't rush — mega-rares drop from all raids anyway
- At 2500 LP: pick Transmutation — resource conversion problems largely solved

### Phase 4 — Unlock Morytania (~2500–5000 LP)

**Goal:** Unlock Morytania; hit T5 (5000 LP)

- Morytania unlock priority: Barrows immediately — semi-AFK, strong mid-game gear
- Grotesque Guardians echo: kill it for 4-tick earth/body staff (magic weapon — upgrade your arsenal)
- ToB: attempt when gear is ready (Barrows + echo weapons should be enough for entry-level)
- Slayer in Slayer Tower: high bone density — prepares well for Soul Harvest
- At 5000 LP: pick Soul Harvest — all combat bones now passively convert to Prayer XP and RC essence

### Phase 5 — Mid-Game Consolidation (~5000–8000 LP)

**Goal:** Hit T6 (8000 LP), unlock Grimoire

- Push Kourend and Morytania hard tasks for bulk LP
- Alchemical Hydra Slayer tasks — high XP + points
- ToB Hard Mode when team or solo mechanics are comfortable (all party members get drops)
- CoX: start attempting if comfortable with the mechanics
- At 8000 LP: pick Grimoire — **NOW the full magic arsenal is online**
  - Switch to Ancient Magicks freely
  - Augury and Rigour unlocked immediately
  - All spells available regardless of region or quest requirements

### Phase 6 — Grimoire + Thermy Echo Active (~8000–16000 LP)

**Goal:** Hit T7 (16000 LP)

- Full magic build now live: Thermy echo weapons + Grimoire + Kandarin task completion
- Run Slayer with Ancient Magicks, burst/barrage tasks for fast points and XP
- Push hard/elite tasks across all 3 unlocked regions
- AFK: Barbarian Fishing, Sulphur Nagua, Gem Crab all continue in background
- At 16000 LP: pick Reloaded → select Abundance — now +10 all non-combat skills and passive coins from every XP drop

### Phase 7 — Endgame Push (~16000–25000 LP)

**Goal:** Hit T8 (25000 LP), unlock Minion

- Push elite and master tasks in Kourend (most task-dense), Kandarin, and Morytania
- ToB HM for bulk drops and points
- Alch Hydra for Slayer XP
- Raids as available
- At 25000 LP: pick Minion — Imp companion active, auto-attacks and auto-loots from here

### Phase 8 — Full Build Online (25000+ LP)

- Complete remaining hard/elite/master tasks for league score
- Late-game bosses: Nightmare, Araxxor, Guardians of the Rift
- CoX + ToB HM for collection log progress
- Fortis Colosseum when Varlamore pushes (hard content)

## Tradeoffs and Warnings

- **Morytania vs Desert as 3rd region:** Morytania chosen for magic synergy (Grotesque Guardians earth staff + Grimoire covers Desert redundantly). Desert has more raw LP from DT2/GOTR tasks. If Grimoire feels optional, reconsider Desert.
- **Endless Harvest vs Abundance T1:** Endless Harvest is pure AFK-first. If Matthew ends up more actively playing than expected, Abundance would have been better for balanced PvM. This is locked at character creation.
- **Minion vs Flask T8:** Flask is higher DPS ceiling but requires skill to time correctly and has zero passive value. Minion is the correct choice given work-AFK playstyle and mid-level experience.
- **ToB readiness:** ToB is mechanically demanding. Have Barrows gear + Thermy echo weapons before attempting. Don't rush it for points.
- **Perilous Moons:** Requires 48 Slayer. With Reloaded → Abundance at T7 this becomes 38 Slayer retroactively. Early game you may need to wait on Moons.
